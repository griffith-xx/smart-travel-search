# 🌟 Smart Travel Search

> Intelligent Thailand Attractions Discovery Platform

A smart travel recommendation system that helps users discover perfect destinations in Thailand through advanced search, personalized recommendations, and dynamic filtering.

## ✨ Features

- 🔍 **Smart Search** - Advanced filtering with tags, categories, and attributes
- 🎯 **Personalized Recommendations** - Learn user preferences and suggest relevant attractions
- 📍 **Location-Based Discovery** - Find attractions by province, region, or nearby locations
- 🏷️ **Dynamic Tagging System** - Flexible attribute management for attractions
- 📊 **User Behavior Tracking** - Improve recommendations based on user interactions
